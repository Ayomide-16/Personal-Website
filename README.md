## ğŸ’» Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites

-   Node.js (v16 or higher)
-   npm or pnpm

### Installation

1.  **Clone the repository**

    ```bash
    git clone https://github.com/Ayomide-16/Personal-Website.git
    cd Personal-Website
    ```

2.  **Install dependencies**

    ```bash
    npm install
    # or
    pnpm install
    ```

3.  **Start development server**

    ```bash
    npm run dev
    # or
    pnpm dev
    ```

    The app will run at `http://localhost:5173` (or similar).

## ğŸ“ Configuration
